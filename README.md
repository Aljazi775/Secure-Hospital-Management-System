## Project Description
The Hospital Management System is a desktop application designed to manage hospital operations. It supports three user roles:
- **Receptionist:** Manages patient registration and appointments.
- **Doctor:** Manages clinical records and diagnoses.
- **Admin:** Manages staff accounts and audit logs.
